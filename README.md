# React Dashboard Frontend

## Project Structure
```
src/
├── components/
├── context/
```

## Available Scripts
- `npm start`: Runs development server
- `npm build`: Creates production build
- `npm test`: Runs tests

## Styling
Uses Tailwind CSS with custom configurations in `tailwind.config.js`


## Set up frontend:
   ```bash
   npm install
   npx create-react-app 
   npm start
   ```
